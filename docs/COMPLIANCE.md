## Data Breach Response

1. Contain — revoke affected credentials immediately
2. Assess — determine scope within 24 hours
3. Notify OAIC — within 30 days if likely to cause serious harm (NDB scheme)
4. Notify affected users — as soon as practicable
5. Document — full incident report in compliance_audit_log
6. Review — post-incident security review within 14 days
